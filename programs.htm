Functions..Methods..Scripts..

Heres a few fun functions to implement in any programming language you like. These problems are meant for fun/practice for a person new to programming. I will slowly add more difficult problems as time progresses. Some of these problems are classic questions I was asked during my undergraduate degree and some I have found cruising the internet.

Fibonacci

Implement a function to compute the nth fibonacci number.

int nthFib(int n){
if(n == 1 || n == 2){
return 1;
}
return nthFib(n-2) + nthFib(n-1);

String Manipulation

Implement a function to reverse a String both iteratively and recursively.

Iteratively

String reverse(String input){
String newString = "";
for(int i = input.length()-1;i>=0;i--){
newString += input.charAt(i);
}
return newString;
}

Recursively

String reverseRec(String input){
if(input.length() == 1){
return input;
}
return reverseRec(input.substring(1)) + input.charAt(0);
}
