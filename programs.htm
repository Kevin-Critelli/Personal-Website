Functions..Methods..Scripts..

Heres a few fun functions to implement in any programming language you like. These problems are meant for fun/practice for a person new to programming. I will slowly add more difficult problems as time progresses. Some of these problems are classic questions I was asked during my undergraduate degree and some I have found cruising the internet.

Fibonacci

Implement a function to compute the nth fibonacci number.

int nthFib(int n){
if(n == 1 || n == 2){
return 1;
}
return nthFib(n-2) + nthFib(n-1);

String Manipulation

Implement a function to reverse a String both iteratively and recursively.

Iteratively

String reverse(String input){
String newString = "";
for(int i = input.length()-1;i>=0;i--){
newString += input.charAt(i);
}
return newString;
}

Recursively

String reverseRec(String input){
if(input.length() == 1){
return input;
}
return reverseRec(input.substring(1)) + input.charAt(0);
}

Prime numbers

Implement a function to compute the nth prime number.

boolean isPrime(int num){
if(num < 2){
return false;
}
for(int i = 2; i <= num / 2; i ++){
if( (num%i) == 0){
return false;
}
}
return true;
}

int nthPrime(int n){
int i = 2;
int counter = 0;
while(true){
if(isPrime(i)){
counter++;
if(counter == n){break;}
}
i++;
}
return i;

Palindromes

A palindrome is a word, phrase, or sequence of symbols that is read the same forward as it is backwards. For example, 'A Santa at Nasa', 'Anna', and 'Cigar? Toss it in a can. It is so tragic' are all palindromes.

Implement a function that when given a string, determins if it is a palindrome or not.

boolean isPalindrome(String input){
int i = 0;
int j = input.length()-1;
while(i < j){
if(input.charAt(i) != input.charAt(j)){
return false;
}else{
i++;
j--;
}
}
return true;
}

Numbers

Implement a function to reverse an integer, do not convert it to a string, use math operations.

int reverseInt(int x){
int newNum = 0;
while((x/10) > 0){
newNum = newNum*10;
newNum += x%10;
x = x/10;
}
newNum = newNum*10;
newNum += x;
return newNum;
}

Time

Implement a function which takes in an integer x representing some number of seconds and output a string representation of the time in hours, minutes, and seconds. For eample if x is 3600 than output is "1 Hour 0 Mins 0 Seconds".

String secondsToTime(int x){
String time = "";
int numHours = x / 3600;
int numMins = (x % 3600) / 60;
x = x % 60;
time = numHours + " hours " + numMins + " minutes " + x + " seconds";
return time;
}

Linked Lists

Implement a function to reverse a Linked List. Assume each node only has a data field and next field, the function takes in one parameter, a pointer to the head of the linked list to reverse and returns a pointer to the new list.

Node reverseLinkedList(Node listHead){
Node new_list = null;
while(listHead != null){
Node ptr = listHead;
listHead = listHead.next;
ptr.next = new_list;
new_list = ptr;
}
return new_list;
}
